# Features

* Somewhat customizable.
* Cross platform (Linux, Windows, MacOS).
* 24-bit `color` support with ANSI.
* Edit framebuffer before rendering.
* The ability to display:
  * Cool animations.
  * Interactive Games.
  * Whatever you come up with...
